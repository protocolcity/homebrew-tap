# protocolcity/homebrew-tap

Homebrew tap for the **ProtocolCity** suite.

```bash
brew tap protocolcity/tap
brew install protocolcity
protocolcity found ~/my-city
protocolcity serve --with-engines
# ‚Üí http://127.0.0.1:8801/
```

Packages (PyPI): `protocolcity`, `protocolcity-worklane`, `protocolcity-workforce`.

See [FIRST_RUN](https://github.com/protocolcity/ProtocolCity-BluePrint) / ProtocolCity docs.
